== Description ==

This integration of [LifterLMS](https://wordpress.org/plugins/lifterlms/) with [myCred](https://wordpress.org/plugins/mycred/) plugin helps users to award myCred points to students using [myCred](https://wordpress.org/plugins/mycred/) Point Management System. There are several events on which student can earn or lose points as defined by the admin. 

Points are awarded upon completion of lesson, section, courses and also deducts points if students fails.

= myCred LifterLMS Features =
Here are the most notable features of myCred LifterLMS plugin.

= Hooks For Lifter LMS =
* **Courses:** myCred LifterLMS provides the ability to award points on different events (such as; when student completes a course, completes a track or enrolled in a course.)
* **Quizzes:** LifterLMS has quizzes integrated in the learning management system, a hook is introduced for quiz related events. Points can be rewarded when student passed / failed or complete a Quiz.
* **Lessons:** Lessons Hook can be used when a lesson of a course is completed. As the student completes any specific course they will be awarded points for its completion.
* **Sections:** Using this Hook admn can award Points to the students who have completed a section of a course. 
* **Membership:** myCred LifterLMS has the feature to create a membership-based course that can be accessed by members only. LifterLMS Membership hook integrated with myCred So the Admin can award points Once the student enrolled as a member.
* **Plans:** Points can be awarded when a student purchase a plan from Lifter LMS
* **Achievements:** Once achievement given in LIFTERLMS, the system will auto-generate points and awarded to the respective student. It also Allows earning achievement for different engagements like completing a lesson.
* **Certificates:** Certificates are given in LIFTERLMS for completing courses, now points can also be awarded when a new certificate is assigned to the Student.

= Logs =
* Logs are created on completion of any event in myCred. We have multiple categories of Logs Like Define Users, Create References, Entries and Points

= Rewarding Points =
* Points can be awarded for different scenarios like on course completion, unlocking an achievement, enrolling into the course and vice-versa.

= Deducting Points =
* It Provides you a feature to deduct points from Student Balance if they fail in a quiz, by defining the value in negative.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred LifterLMS
3. Find myCred LifterLMS and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
